### 복잡도 ### 

복잡도는 알고리즘의 성능을 나타내는 척도이다. 

시간 복잡도
 - 알고리즘 실행 시간
 - 알고리즘을 위해 필요한 연산의 횟수
 - 메모이제이션 ( 메모리 소비를 증가 시키고 시간을 단축 시키는 방법)
 - 시간 복잡도를 표현 할때는 빅오 표기법을 사용 한다. 

공간 복잡도
 - 특정한 크기의 입력에 대해 알고리즘이 얼마나 많은 메모리를 차지 하는지 의미한다. 
 - 알고리즘을 위해 필요한 메모리의 양

예시)
array = [3, 5, 1, 2, 4] # 5개의 데이트( N = 5)
summary = 0

for X in array:
    summary += X

print(summary)
